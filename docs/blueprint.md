# **App Name**: The Chronicle

## Core Features:

- Landing Page: Showcase featured stories and brief news summaries.
- Main News Page: Display the latest news articles, organized by category.
- Article Page: Dedicated page for each article, including text formatting.
- Category Navigation: Allow users to filter articles by category.
- Search Functionality: Enables users to search for specific news articles.
- Theme Toggle: Enable the toggle between dark and light mode
- Article Sharing: Provide an option for users to share articles.

## Style Guidelines:

- Primary color: Deep orange (#C4540A) to evoke reliability.
- Background color: Very light grey (#F2F0EB) provides a clean, readable canvas.
- Accent color: Soft orange (#D9691F) for highlighting key elements without overwhelming the user.
- Headline font: 'Playfair', serif. Intended for headlines to create an elegant, literary feel.
- Body font: 'Inter', sans-serif. Intended for body text for modern and neutral readability.
- Use 'Lucide React' icons, emphasizing simplicity and clarity for intuitive navigation.
- Employ a responsive grid layout optimized for readability and visual hierarchy across devices.
